# Gemini 3 Pro Redesign - Step by Step Guide

## Overview
This directory contains prompts for using Gemini 3 Pro to redesign the English Learning App. Each step focuses on a specific screen or component.

## Process
1. I will provide you with a detailed prompt for each step
2. You copy the prompt and send it to Gemini 3 Pro (via Google AI Studio or API)
3. You share Gemini's response with me
4. I implement the changes in the app
5. We test and move to the next step
